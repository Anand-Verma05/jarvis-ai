# Jarvis AI Voice Assistant 🤖

Jarvis is a Python-based AI voice assistant that can perform various tasks using voice commands.  
It uses speech recognition and text-to-speech to interact with the user and execute commands such as opening websites, searching the internet, and performing system operations.

---

## 🚀 Features

- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Open websites like Google, YouTube, etc.
- 🔍 Perform web searches
- 🖥 Execute basic system commands
- ⏰ Tell time and respond to basic queries

---

## 🛠 Tech Stack

- **Python**
- **SpeechRecognition**
- **pyttsx3**
- **PyAudio**
- **Webbrowser**
- **OS Module**
